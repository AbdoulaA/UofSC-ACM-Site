# ACM UofSC Website

Official website for the Association for Computing Machinery (ACM) at the University of South Carolina Molinaroli College of Engineering and Computing.

This platform serves as the central hub for ACM members, showcasing events, leadership, initiatives, and resources while providing an extensible foundation for future features.

---

## Overview

This project is a modern, component driven single page application built to support:

- Event and member management
- Organization updates and announcements
- Secure admin workflows
- Scalable frontend architecture
- Clean, maintainable UI systems

The architecture emphasizes modularity, performance, and long term maintainability.

---

## Tech Stack

- Vite  
- TypeScript  
- React  
- shadcn-ui  
- Tailwind CSS  
- React Router  

---

## Getting Started

### Prerequisites

- Node.js (recommended via nvm)
- npm

### Installation

```sh
# Clone the repository
git clone https://github.com/<your-username>/acm-uofsc-website.git

# Navigate into the project directory
cd acm-uofsc-website

# Install dependencies
npm install

# Start development server
npm run dev
